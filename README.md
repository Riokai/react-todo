# React-Todos
基于React搭建的备忘录

使用到的工具：
* [Webpack](https://github.com/webpack/webpack): 项目开发、打包
* [Babel](https://github.com/babel/babel): 拥抱ES6
* [Mocha](https://github.com/mochajs/mocha): 测试框架
* [Enzyme](https://github.com/airbnb/enzyme): React组件单元测试

### 运行
```
npm install
npm start
```

### 测试
```
npm test
```
